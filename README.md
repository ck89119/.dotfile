# .dotfile

It's my .dotfile for Unix/Linux System.

## Installation

### one-step:
    curl -fsSL https://raw.github.com/ck89119/.dotfile/master/auto-install.sh  

### or manually:

	git clone https://github.com/ck89119/.dotfile.git ~/.dotfile 
	cd ~/.dotfile
	./install.sh
